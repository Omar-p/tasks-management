1. Introduction

This document provides comprehensive API documentation for the Tasks Management system. The API allows users to manage their tasks, authenticate, and manage their profiles.

1.1. Base URL

/api

1.2. Authentication

Most endpoints require authentication via JWT Bearer token. Include the token in the Authorization header: Authorization: Bearer {access_token}
